Overview of World Wide Web

- World Wide Web: is a collection of HTML documents.
Note: pdf documents, word documents, excel documents won't work.
- HTML stands for: Hyper Text Markup Language.
-Webpag links: Different webpages are linked or glued together using hyperlinks e.g click me.Will take you to www.webpagelink.com

Major Pieces of the Web

There are four main components of the world wide web as follow:

1. User:This is you, sitting on a computer right now.
2. Computer: This is the machine you will use that would be having a browser (e.g Internet Explorer, Firefox, Google Chrome, Safari etc.).
3. Internet: This is the actual structure that will direct the request path to the actual destination.
4. Server: This is a computer known as server that only serve information requested by the user.

Note: a protocol known as http (hyper text tranfer protocol) is used to communicate from the user's computer via web browser through the internet path to the Server'.The server communicate back with requested information.

HTML Tags

- Tags are also used to make links to images, webpages etc.
